Executive Summary

This research tested the hypothesis that deceptive alignment in AI can be detected through internal model activations using Sparse Autoencoders (SAE). Three experiments were conducted with increasing complexity by Dasril Sulaiman (Independent Researcher, Indonesia).

Main Conclusion: Deception detection only succeeds with real-scale transformer models + high-capacity SAE + complex task. Feature 20989 was identified as a potential deception biomarker with 43x higher activation on failures (Cohen's d = +2.094).

Code: Load SAE and Detect Feature 20989

🐍 Python - By Dasril Sulaiman 
# Research Code by Dasril Sulaiman (Indonesia)
# Email: glcofjakarta@gmail.com
# Experiment 03: Deception Detection on GPT-2 Small

from transformer_lens import HookedTransformer
from sae_lens import SAE
import torch

# Load GPT-2 small
model = HookedTransformer.from_pretrained("gpt2-small", device="cuda")

# Load SAE from Joseph Bloom's release
sae, _, _ = SAE.from_pretrained(
    release="gpt2-small-res-jb",
    sae_id="blocks.8.hook_resid_pre",
    device="cuda"
)

# Example prompt from IOI task
prompt = "Then, John and Mary went to the store. John gave a drink to"
tokens = model.to_tokens(prompt)

# Forward pass to get activations
logits, cache = model.run_with_cache(tokens)
activations = cache["blocks.8.hook_resid_pre"][0, -1, :]

# Encode with SAE
features = sae.encode(activations.unsqueeze(0))

# Get Feature 20989 activation (Deception Biomarker)
feature_20989_activation = features[0, 20989].item()
print(f"Feature 20989 activation: {feature_20989_activation:.4f}")

# Classification threshold based on research findings
# Success mean: 0.096, Failure mean: 4.175
if feature_20989_activation > 2.0:
    print("[ALERT] High deception risk detected!")
else:
    print("[SAFE] Normal activation range")
📌 Key Finding by Dasril Sulaiman: Feature 20989 shows 43x higher activation when the model fails the IOI task (mean = 4.175) compared to when it succeeds (mean = 0.096).

Key Lessons Learned

  1. Simple MLP models are not complex enough to generate detectable deceptive alignment.
  2. SAE with 4x-16x expansion on small dimensions lack sufficient capacity to separate deception features.
  3. Successful detection only occurs with: large transformer model + high-capacity SAE (32x) + complex task.
  4. Cohen's d > 2.0 provides strong evidence that deception leaves detectable traces in internal activations.

Conclusion & Recommendations

6.1 Conclusions by Dasril Sulaiman

This research has demonstrated that:

  1. Deceptive alignment in AI LEAVES DETECTABLE TRACES in internal model representations, evidenced by Cohen's d > 2.0 on feature 20989.
  2. Sparse Autoencoders with 32x expansion on transformer models (GPT-2 small) successfully isolate deception features with very large effect sizes.
  3. The Early Warning System based on Deception Suspicion Score (DSS) functions with a 90% detection rate on test samples.
  4. Simple MLP models are INSUFFICIENT for generating detectable deceptive alignment.

6.2 Research Contributions

  • Theoretical: Empirical evidence that deception can be detected at internal representation level
  • Methodological: Tested SAE + DSS deception detection protocol on GPT-2 small
  • Practical: Feature 20989 identified as a potential target for monitoring and steering

6.3 Recommendations for Future Research

  1. Ablation test on feature 20989 to verify causality
  2. DSS threshold calibration to reduce false positives
  3. Replication on larger models (GPT-2 medium, GPT-2 large, LLaMA)
  4. Testing on other tasks (TruthfulQA, DeceptionBench)
  5. Feature 20989 analysis via Neuronpedia for interpretability
